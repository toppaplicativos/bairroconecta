
# BairroConecta 🏠🤝

**BairroConecta** é uma plataforma comunitária integrada, desenvolvida para fortalecer os laços entre moradores e o comércio local. O aplicativo funciona como um hub central para todas as necessidades do bairro, desde a busca por um novo lar até a comunicação direta com a ouvidoria municipal.

## 🚀 Funcionalidades Principais

- **🏠 Imobiliária Local**: Listagem refinada de propriedades para venda e aluguel com filtros inteligentes.
- **🛍️ Guia Comercial**: Diretório completo de lojas e serviços do bairro, organizado por categorias.
- **🍕 Alimentação (Delivery)**: Portal gastronômico para descobrir e pedir comida dos melhores lugares da vizinhança.
- **🏛️ Mural da Ouvidoria**: Canal de transparência para registrar e acompanhar demandas públicas (iluminação, buracos, etc.) com auxílio de IA.
- **🏥 Portal da Saúde**: Agendamento de consultas e assistente de triagem inteligente baseado em IA.
- **💬 Fórum da Comunidade**: Discussões locais e enquetes interativas para engajamento dos moradores.
- **🤖 Assistente Local de IA**: IA treinada para responder qualquer dúvida sobre o bairro e serviços disponíveis.

## 🎨 Identidade Visual

- **Cor Primária**: Soft Blue (#77B5FE) - Transmite confiança e serenidade.
- **Background**: Light Gray (#F0F4F8) - Visual moderno e limpo.
- **Destaque**: Green (#90EE90) - Para ações positivas e destaques.
- **Tipografia**: PT Sans - Uma fonte sans-serif humanista para legibilidade e proximidade.

## 🛠️ Tecnologias Utilizadas

- **Frontend**: Next.js 15 (App Router), React, Tailwind CSS.
- **UI Components**: Shadcn UI, Lucide Icons.
- **Backend/Database**: Firebase Firestore, Firebase Authentication, Firebase Storage.
- **Inteligência Artificial**: Genkit (Google AI / Gemini).
- **Mapas**: React Map GL / Mapbox.

## 📦 Como Instalar e Rodar

1. Clone o repositório:
   ```bash
   git clone https://github.com/SEU_USUARIO/bairroconecta.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Configure as variáveis de ambiente em um arquivo `.env` com suas chaves do Firebase e Gemini.
4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
