# **App Name**: BairroConecta

## Core Features:

- Property Listings: List properties for sale or rent in the neighborhood.
- Local Business Directory: Directory of local businesses, divided by categories.
- Community Forum: A forum for residents to discuss local issues.
- AI Neighborhood Assistant: AI assistant to answer user questions about the neighborhood; uses a tool that decides whether and how to incorporate structured information.
- Events Calendar: Calendar of events in the neighborhood.
- Classified Ads: Classifieds section for buying, selling, or trading items.

## Style Guidelines:

- Primary color: Soft blue (#77B5FE) to convey trust and community.
- Background color: Light gray (#F0F4F8) to create a clean and modern look.
- Accent color: Green (#90EE90) for calls to action and highlights.
- Body and headline font: 'PT Sans', a humanist sans-serif, will be used throughout the app.
- Use simple, clear icons for navigation and modules.
- Mobile-first design with a clear bottom navigation bar.
- Subtle transitions and animations to enhance user experience.
